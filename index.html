Bienvenue

Ce site a pour but de vous presentez et vous apprendre les bases du gLua, language pour le jeu Garry's Mod.

Le gLua est un language de programmation assez paticulier, puis qu'il est prope au jeu vidéo Garry's Mod développé par FacePunch studio, dont le créateur est Garry Newman. Ce language est tès complet et permet de modeler le jeu comme bon nous semble.
